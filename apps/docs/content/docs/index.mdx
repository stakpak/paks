**Paks** is a CLI tool for scaffolding, installing, publishing, and managing agent skills across different AI coding agents including Claude Code, Cursor, VS Code, Goose, OpenCode, and Stakpak.

## Quick Start

```bash
# Create a new skill
paks create my-skill

# Validate skill structure
paks validate ./my-skill

# View skill details
paks info ./my-skill

# List installed skills
paks list

# Publish (dry-run)
paks publish ./my-skill --dry-run
```

## Installation

```bash
# Build from source
cargo build -p paks-cli --release

# Binary location
./target/release/paks
```
